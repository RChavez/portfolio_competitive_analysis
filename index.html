Website THREE: Trent Walton
URL: trentwalton.com
Date Visited: January 17th, 2012
Brand: Bold- The visual objects of the inital site are large and very noticeable (see image six)
Clean- The few elements that are on the site make the page seem clear of unecessary clutter.
Texture- The author presents his own brand by adding a themed background related to his blog.
Functionality:

Site uses a Three Box layout. The site is simple in design, but poorly displays it's content due to it's inability to attract attention in an attractive fashion. Upon investigating secondary pages (i.e. Individual blog posts), the user will find that there are multiple designs for the site- the only saving piece of this choice is that the same, simple layout is used throughout (see Image seven).

On one of the secondary pages, the user can utilize a search bar to enter keywords that returns related posts.
Navigation:

The site was easy to navigate, but the user may find it difficult to determine how much content is available within the site.

Website FIVE: David Gamache's Blog
URL: davegamache.com
Date Visited: January 17th, 2012
Brand: Minimal- There are very few differing elements ever on a single page, making the site appealing.
Structured- Every page is neatly structured so the user never feels lost and confused.
Clean- Design is neat and tidy, easy to understand and readily accessible.
Functionality:

This blog takes on a variation of the Featured Photo layout (see image ten), showcasing a simple above-the-fold lead along with the text beneath the lead. The simplicity makes the site excitingly functional wherever the the user travels within the site.
Interactivity:

The blog restricts interactivity in order to add to its simple elegance. A simple fade animation occurs with the top photo once the viewer scrolls below the fold. The only clickable links are those found in the header and footer, which only navigate to a list of the author's articles (see image eleven), a bio page, and the home-page. No fillable forms are present.
Navigation:

The site is extremely easy to navigate. The simplicity makes reading the author's content enjoyable. Navigation is so simple that the inclusion of a site map would be a waste. The site essentially only contains 4 pages. The home-page, the list of articles, an about page, and the current article being read page.

Website SIX: Kicksend
URL: kicksend.com
Date Visited: January 18th, 2012
Brand: Flashy- The visual elements within the page "pop", identifying it as a hip and casual service.
Simple-

Stylistically, there is not a large amount of detail involved, which allows for minimal distraction from what the creator's want the viewers to see.


Organized-

The visual elements below the fold are organized into noticable grids, showing that the service is still serious about conducting business.
Functionality:

The site uses a Featured Graphic layout (see image twelve). The site loads few but large visual elements which may be too dramatic for some site visitor; however, the small amount of textual information balances the site by showing it's simplicity. Site features a "Connect with Facebook" option which (in my opinion) takes away from it's uniqueness. Being a service based website, the Facebook, Google+, and Twitter share options make it obvious they want the user to spread word of the site as much as possible (see image thirteen).
Interactivity:

The essentials are present- such as hotlinks which animate attractively. Another obvious is the sign up fill forms which act exactly as how one would expect. Also included are icon animations which are triggered by mouse-over. The personality of the website is accented by these animations by making it seem fun and dynamic.
Navigation:

The site contains an average amount of secondary pages, which include the usual About, FAQ, and Contact pages. Navigation as a whole is simple, with little need for any navigational need. The design changes between primary and secondary pages which may make it challenging for some to use.
