Website TWO: Michela Chiucini -Web Design-
URL: http://www.colazionedamichy.it/
Date Visited: January 17th, 2012
Brand: Aged- The site goes for a vintage, retro appearance.
Textured- The site also uses a paper-like background to add a tactile presentation.
Light-hearted- The bright colors make the site playful and convey a positive attitude.
(See image 4)
Functionality:

The website loads with a good balance of visual and textual information. All seondary pages continue with the same design making it simple to navigate around and still connected to the same site.
Interactivity:

The site uses constant hotlinking along with hover effects that cue the user when additional informaiton can be found. Compared to other sites visited, this one does not bombard the user with clickable links. The site also uses forms to submit comments questions about the work that the creator has done. (See image five)
Navigation:

The site was very easy to navigate due to the consitency with each page and the overall design. There was no hidden content; in fact, there was little content which made the site easy to navigate.

Website THREE: Trent Walton
URL: trentwalton.com
Date Visited: January 17th, 2012
Brand: Bold- The visual objects of the inital site are large and very noticeable (see image six)
Clean- The few elements that are on the site make the page seem clear of unecessary clutter.
Texture- The author presents his own brand by adding a themed background related to his blog.
Functionality:

The site is simple in design, but poorly displays it's content due to it's inability to attract attention in an attractive fashion. Upon investigating secondary pages (i.e. Individual blog posts), the user will find that there are multiple designs for the site- the only saving piece of this choice is that the same, simple layout is used throughout (see Image seven).
Interactivity:

Becuase it is a personal blog, there is minimal interactivity on the site. The user clearly illustrates that the site is strictly designed for his postings. There are few links present which only reference commennts by other viewers and hotlinks which lead to more information about a single post. On one of the secondary pages, the user can utilize a search bar to enter keywords that returns related posts.
Navigation:

The site was easy to navigate, but the user may find it difficult to determine how much content is available within the site.
